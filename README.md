## Deployment
Follow the steps to deploy in your workstation.

Software required:

-nodejs v0.10.40

-npm v1.4.28

* Checkout project
```sh
$ npm install
```

```sh
$ bower install
```
* If you face issues running "bower install", try first to run this:
```sh
$ npm install -g bower
```
* To run the application:
```sh
$ node app.js
```
* now go to http://localhost/3000/
