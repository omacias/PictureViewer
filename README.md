## Features and Stuff
* NodeJs
* AngularJs
* Boostrap
* Font Awesome
* JQuery

## Deployment
Follow the steps to deploy in you workstation.
Software required:
-nodejs v0.10.40
-npm v1.4.28

* Checkout project
* run "npm install"
* run "bower install" (If you face issues running this command, try first to run "npm install -g bower" and try again)
* run "node app.js"
* now go to http://localhost/3000/
